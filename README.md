## ELENA PHOTOGRAPHY

A single page photography website for practice while learning HTMl and CSS

### AUTHOR
MaryAnne Wanjeri 
### RELEVANCE OF THE PROJECT
The website is a portfolio for a photographer advertising their work.

It shows past evidence of their work,what their motivation is and how an interested party can reach them

###  STRUCTURE OF THE PROJECT
The webpage includes: 
    -a styled name-logo
    -hero section of the photographer 
    -additional information about the photographer(e.g their motivation)
    -Evidence of previous work by the photographer
    -A few journal entries
    - Contact details
